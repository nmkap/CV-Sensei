09 · Motion & Transitions prefers-reduced-motion override applied globally · always use CSS vars
Fast · 150ms
Hover colour changes, icon swaps, focus ring appearance. Immediate feedback.
--t-fast: 150ms ease
Normal · 200ms
Button hover states, card lift, border transitions. Standard interactions.
--t-normal: 200ms ease
Slow · 300ms
Page load reveals, modal open/close, sidebar expand. Considered transitions.
--t-slow: 300ms ease
The global @media (prefers-reduced-motion: reduce) rule at the top of this stylesheet overrides all animations and transitions to 0.01ms for users who have enabled reduced motion in their OS accessibility settings. This covers the chat typing animation, carousel scroll, page-load reveals, and button hover lifts.

/* ── CV Sensei Design Tokens v2 ── */
:root {
  /* Core palette */
  --c-dark: #222831; /* sidebar, hero, headings · 14.83:1 AAA */
  --c-mid: #6B6560; /* body text, borders · 5.74:1 AA ✦ was #393E46 */
  --c-accent: #B55400; /* CTAs, links — LIGHT BG ONLY · 4.96:1 AA */
  --c-accent-dark: #E07A15; /* icons/text on #222831 ONLY · 4.92:1 AA */
  --c-surface: #F2F0EC; /* page bg, cards ✦ replaces #EEE + #FFF */
  --c-white: #FFFFFF; /* elevated cards, chat bubbles */
  --c-border: #E0DCD6; /* dividers, input borders */

  /* Text */
  --c-text-primary: #222831; /* 14.83:1 AAA */
  --c-text-body: #3D3935; /* 11.92:1 AAA */
  --c-text-secondary: #6B6560; /* 5.74:1 AA */
  --c-text-muted: #625D58; /* 6.51:1 AA ✦ was #8A857F (fail) */

  /* Semantic */
  --c-success: #2E7D52;  --c-error: #C0392B;  --c-warning: #C8860A;  --c-info: #2C5282;

  /* Fonts */
  --font-display: 'Playfair Display', Georgia, serif;
  --font-ui: 'Lato', system-ui, sans-serif;

  /* Spacing (4px grid) */
  --sp-1: 4px;  --sp-2: 8px;  --sp-4: 16px;  --sp-6: 24px;  --sp-8: 32px;  --sp-12: 48px;

  /* Radius */
  --r-sm: 6px;  --r-md: 10px;  --r-lg: 14px;  --r-xl: 20px;  --r-pill: 999px;

  /* Focus rings */
  --focus-light: 0 0 0 3px rgba(181,84,0,0.30); /* on light bg */
  --focus-dark: 0 0 0 3px rgba(224,122,21,0.50); /* on dark bg */

  /* Transitions */
  --t-fast: 150ms ease;  --t-normal: 200ms ease;  --t-slow: 300ms ease;
}
